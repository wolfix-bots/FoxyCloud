# 🦊 FoxyCloud

> Deploy the **Webfoxy** WhatsApp bot in seconds.

FoxyCloud is the official deployment dashboard for [Webfoxy](https://github.com/wolfix-bots/Webfoxy). Paste your session ID, click deploy — your bot is live on Render.

## ✨ Features
- 🚀 One-click Render deploy
- 🔒 Session ID stored locally (never sent anywhere)
- 🌙 Dark neon UI with animated particles
- 📱 Fully mobile-responsive

## 🚀 Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/wolfix-bots/FoxyCloud)

## 🛠️ Development

```bash
npm install
npm run dev
```
